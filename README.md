# Multiple Linear Regression using OLS (Closed Form)

## Overview

This project implements Multiple Linear Regression using the Ordinary Least Squares (OLS) closed-form solution on the California Housing dataset.

## Features

* Data Cleaning and Preprocessing
* Feature Engineering (Encoding + Scaling)
* OLS Closed Form Implementation (from scratch using NumPy)
* Model Evaluation (RMSE, R² Score)
* Visualization (Actual vs Predicted, Residual Plot)

## Dataset

California Housing Dataset

## Project Structure

```

data/raw/
data/processed/
notebooks/mlr_ols_project.ipynb


outputs/plots/
outputs/results/
```

## Results

* RMSE and R² Score computed
* Visual analysis performed using scatter and residual plots
